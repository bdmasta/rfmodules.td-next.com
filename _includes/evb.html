EVB features

The TD modules Evaluation Board provides a development and demonstration platform for Telecom Design TD SIGFOXTM gateway modules and software tools.

The EVB breaks all available module pins on standard pitch headers, along with a convenient SMA connector for plugin a common half wave swivel antenna.

The EVB can be connected to a PL using an included USB serial adapter cable.
